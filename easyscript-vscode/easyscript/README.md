# EasyScripts VS Code Extension

This is the official VS Code extension for **EasyScripts**, a natural-style JavaScript wrapper language.

## Features

- Syntax highlighting for EasyScripts
- Built-in themes
- Basic editor support

## Installation

### From VSIX (local)
1. Download `easy-scripts-vscode-0.1.0.vsix`.
2. Open VS Code → Extensions → `Install from VSIX`.
3. Select the `.vsix` file.

### From Marketplace (future)
Search for `EasyScripts` in the VS Code Extensions Marketplace.
